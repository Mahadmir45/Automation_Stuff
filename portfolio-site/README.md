# Portfolio Site

Single-page portfolio website showcasing automation engineering projects with interactive visualizations.

## Includes

- project cards for all five major automation projects
- KPI stats summary section
- SVG bar chart for estimated hours saved
- Canvas radar chart for skill coverage
- responsive modern UI with clean gradients and glass-style panels

## Run Locally

Open `index.html` directly in your browser, or serve with any static server.

## GitHub Pages Deployment

This repository includes `.github/workflows/deploy-portfolio.yml` to deploy `portfolio-site` automatically on pushes to `main`.

Expected site URL:

- `https://mahadmir45.github.io/Automation_Stuff/`

## Custom Domain Ready Setup

1. Copy `CNAME.example` to `CNAME`.
2. Replace the domain with your real domain (for example, `portfolio.mahad.dev`).
3. Commit and push `portfolio-site/CNAME`.
4. In GitHub repository settings, add the same domain under Pages.
5. Point your DNS provider CNAME record to `mahadmir45.github.io`.
